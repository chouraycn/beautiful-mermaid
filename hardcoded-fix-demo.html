graph TD
    Start([开始]) --> Input[输入账号密码]
    Input --> Validate{验证格式?}
    Validate -->|失败| ShowError[显示错误提示]
    ShowError --> Input
    Validate -->|成功| CheckCode{验证码?}
    CheckCode -->|需要| VerifyCode[校验验证码]
    VerifyCode -->|失败| ShowCodeError[验证码错误]
    ShowCodeError --> Input
    VerifyCode -->|成功| Auth[调用认证服务]
    CheckCode -->|不需要| Auth
    Auth -->|失败| ShowAuthError[认证失败]
    ShowAuthError --> Input
    Auth -->|成功| GenerateToken[生成Token]
    GenerateToken --> SaveToken[存储Token]
    SaveToken --> Success([登录成功])
